# ASCIIcam - Kameranızı ASCII Sanatına Dönüştürün

Canlı kamera akışınızı ASCII karakterlerine dönüştüren interaktif bir web uygulaması.

## 🚀 Başlamak

1. `index.html` dosyasını bir web tarayıcısında açın
2. Kamera erişim izni verin
3. Hazır! Canlı ASCII video akışını göreceksiniz

## 🎮 Kontroller

### Panel Kontrolleri
- **Parlaklık**: Video parlaklığını ayarlayın (-50 ~ +50, default: -30)
- **Metin Rengi**: ASCII karakterlerinin rengini seçin (default: yeşil #00ff00)
- **Arka Plan**: Arka plan rengini değiştirin (default: siyah #000000)
- **Dışa Aktar Formatı**: Kaydetme formatını seçin (.txt, .jpg, .png)
- **3s Zamanlayıcı**: Fotoğraf çekerken 3 saniye bekleme süresi ekle
- **photo mode**: Fotoğraf modu (tek karesini yakala)
- **movie mode**: Video modu (canlı akış)

### Klavye Kısayolları
- **SPACE** - Fotoğraf çek
- **S** - Dosya olarak kaydet
- **R** - Tüm ayarları sıfırla
- **M** - Mod değiştir (photo ↔ movie)

## ✨ Özellikler

✅ **Canlı Kamera Akışı** - Gerçek zamanlı ASCII dönüşümü
✅ **Mirror Efekti** - Kamera görüntüsü yatay çevrilir
✅ **Ayarlanabilir Parlaklık** - Video karşıtlığını optimize et
✅ **Özelleştirilebilir Renkler** - Metin ve arka plan renkleri (default: yeşil yazı, siyah arka plan)
✅ **Fotoğraf Modu** - Anlık kareleri yakala
✅ **Video Modu** - Canlı akışı izle
✅ **Çoklu Format Dışa Aktarma** - TXT, JPG, PNG formatlarında kaydet
✅ **Zamanlayıcı Seçeneği** - Fotoğraf çekerken hazırlanma süresi
✅ **Responsive Tasarım** - Tüm cihazlarda çalışır
✅ **Detaylı ASCII Karakterler** - 16 farklı karakter ile yüksek tonlama derecesi

## 📋 Sistem Gereksinimleri

- Modern web tarayıcı (Chrome, Firefox, Safari, Edge)
- Çalışan webcam/kamera
- HTTPS bağlantısı (veya localhost)

## 🔒 Gizlilik

- Tüm işlemler tarayıcınızda gerçekleşir
- Hiçbir veri sunucuya gönderilmez
- Kameranızı yalnızca siz kontrol edersiniz

## 💡 İpuçları

1. **En iyi sonuçlar için**: İyi aydınlatılmış bir ortamda kullanın
2. **Karakter ölçeğini ayarla**: Küçük değerler daha iyi detay verir
3. **Parlaklığı deneyeyin**: Gölgeli alanlar için parlaklığı artırın
4. **Video modu**: Gerçek zamanlı eğlence için
5. **Fotoğraf modu**: Kaliteli kareleri kaydetmek için

## 🛠️ Geliştirici Notları

- **ASCII karakterler**: ` .,;:=|!/+*#%@█` (16 farklı karakter)
- **Default parlaklık**: -30 (daha koyu görünüm)
- **Default metin rengi**: Yeşil (#00ff00)
- **Default arka plan**: Siyah (#000000)
- **Mirror efekti**: Canvas'ın yatay çevrilmesi ile sağlanır
- **Responsive tasarım**: Flex layout kullanılmıştır
- **En boy oranı**: Kamera feed'in oranı korunur
- **Tüm modern tarayıcılarda uyumludur
